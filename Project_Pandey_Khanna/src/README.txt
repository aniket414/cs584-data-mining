Steps to run the code:
1.	Download Infographic_Visualization.ipynb and Models.ipynb file
2.	Open JupyterLab on Anaconda Navigator or Google Colab Notebook
3.	On the menu bar click File -> Open -> Filename
4.	Keep the ipynb file and Dataset folder in the same directory
5.	One by one run each cell in the notebook

The source code along with all the commits can be found on https://github.com/aniket414/car-insurance-fraud
